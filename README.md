# Java of Empires

Projeto de um jogo simples feito em Java.  
Os personagens podem se mover, atacar e perder vida. Quando a vida chega a zero, eles desaparecem da tela.

## Funcionalidades

- Movimento dos personagens
- Ataque em área
- Dano baseado no ataque
- Remoção automática de personagens mortos
- Efeito visual ao morrer (fade-out)
